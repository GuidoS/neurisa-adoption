An adoption story

About AppGeo
  • AppGeo been in business for 26 years
  • I have been there for the last 10 years
  • AppGeo is always looking for effective tools and partners
  • AppGeo is currently partners with Google, Carto, and Safe Software (makers of FME)
Best of Breed
  • AppGeo believes in using the best tool for the job
  • AppGeo has used tools from esri, AutoCAD, Bentley, Safe Software, Google, Microsoft, and Blue Marble
  • AppGeo keeps up with multiple vendors and tools tied to specific domains and usecases
  • AppGeo has built tools to allow engineers to work in their CAD environment but also synch their latest data with their web mapping applications
A Visit From Paul Ramsey
  • After hearing Ramsey speak at NSGIC, Micheal Terner invited him to speak to our company
  • Satisfaction from the staff is a selling point for OSS
  • We were sold and were ready to try to make open source tools a part of our work
  • Developers took off right away, the rest of us did not change for a while

Breaking up is hard to do
  • It is not easy to give up the tools you are used to
  • It takes a while to get comfortable with new tools
  • I ended up using a macintosh just to stop myself from using familiar tools
  • I started to learn the how to translate ideas from esri to other open source tools
Satisfaction
  • I am now more equiped to work with most databases
  • I now build tools using basic database concepts
  • I now know how to review data with QGIS
  • I now know how to get started with other tools of interest like R or Spark
